# 技术

最近接触了vue 然后写了一个练手的项目

使用vue-cli脚手架来搭建了的项目

技术： vue2  + vue-router  + ES6 + axios 

框架有 mint-ui(不建议使用，早已经不更新了，推荐vant)，mui
